# Fetcher.io - Zig Library Registry

[![Website](https://img.shields.io/badge/website-live-brightgreen)](https://nucleonit.github.io/fetcherio-zig-deps/)

> A modern, open-source package registry for the Zig programming language.  
> Stop searching for dependencies on GitHub - find them here.

## 📖 What is it?
Fetcher is a lightweight registry where Zig developers can discover, search, and manage libraries. It collects a wide range of packages, similar to resources like `crates.io`, but specifically tailored for Zig.

## 🎯 Who is it for?
This project is designed to improve the Zig programming experience. It helps you find libraries much faster, so you can focus on writing code instead of hunting for repositories.

## 🚀 Features
- **Suggest a new package:** Click the `+` button in the top right corner and fill out the submission form.
- **Update an existing package:** Click the `↻` button to open the update form.
- **Light/Dark Themes:** Tap the toggle in the `⚙` Settings menu to switch between themes.
- **Interactive Cards:** Every package card is a clickable block. Clicking it opens a sidebar with installation instructions and full documentation.

## 🤝 How can I help the project grow?
Your contributions are welcome! You can help by:
- Submitting new packages via the website form.
- Opening an Issue or Pull Request on GitHub to fix bugs or suggest improvements.
- Giving the repository a Star ⭐ if you find it useful!
